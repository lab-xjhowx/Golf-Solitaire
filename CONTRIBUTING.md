## Como contribuir

Obrigado por considerar contribuir com o Golf Solitaire. Este projeto usa LÖVE (Love2D) e Lua.

## Fluxo recomendado

1. Abra um issue descrevendo o problema ou a melhoria.
2. Faça fork do repositório.
3. Crie uma branch a partir de `main` com um nome descritivo.
4. Faça commits pequenos e claros.
5. Abra um Pull Request descrevendo o que foi alterado e o motivo.

## Rodando localmente

Pré-requisitos: LÖVE (Love2D) 11.x instalado.

```bash
git clone https://github.com/lab-xjhowx/Golf-Solitaire.git
cd golf-master
love .
```

## Testes

```bash
lua tests/rules_score_tests.lua
```

## Padrões

- Mantenha o estilo atual do código.
- Evite adicionar dependências externas sem necessidade.
- Prefira mudanças pequenas e revisáveis.
